# weapp-boilerplate

> 微信小程序样板项目，使用Vue单文件组件的开发体验进行小程序开发。

## 克隆到本地

```shell
$ git clone https://github.com/tjeeay/weapp-boilerplate.git

$ cd weapp-boilerplate
```

## 安装依赖

```shell
$ npm i

// or

$ yarn i
```

## 运行Demo

### 开发环境（监听文件自动更新）

```shell
$ npm run dev
```

### 生产环境

```shell
$ npm run build
```
